# SpeedRead AI

Paste anything. AI summarizes it. Read it in minutes.

## Features
- Rolling 3-line speed reader with word pacer
- AI summarization (Anthropic Claude)
- PDF, EPUB, image (OCR), and plain text import
- Syntax highlighting — nouns, verbs, filler words color-coded
- Adjustable WPM and summary length

## Setup
1. Open `index.html` in a browser (or deploy to GitHub Pages)
2. Enter your [Anthropic API key](https://console.anthropic.com/) in the key field
3. Paste text or upload a file and hit **Summarize & Read**

## Deploy to GitHub Pages
1. Create a new repo on GitHub
2. Push this code
3. Go to Settings → Pages → Deploy from main branch
